# Snackis
#### sugar-free co-creation

### Dependencies
Snackis requires a ```C++1z``` capable compiler/standard library to build, and defaults to using ```clang``` with ```libc++```; it further depends on ```libcurl```, ```libsodium``` and ```libuuid``` for core functionality, as well as ```libcurses``` and ```libform``` for the UI.
